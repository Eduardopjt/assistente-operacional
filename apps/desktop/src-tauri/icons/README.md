# Desktop App Icons

Place the following icon files here before building for production:

- `32x32.png` - Standard icon
- `128x128.png` - Standard icon
- `128x128@2x.png` - HiDPI icon
- `icon.icns` - macOS icon bundle
- `icon.ico` - Windows icon

Use a tool like `png2icns` or `electron-icon-builder` to generate from a source PNG.

Recommended source: 1024x1024 PNG with transparent background.
